# 👑 Посібник адміністратора MarketOblik

## Вступ

Адміністратор мережі має можливість:
- Керувати системними вкладками (додавати/видаляти відділи)
- Налаштовувати акції на товари
- Керувати доступом до API
- Відстежувати активність магазинів

## 🔐 Отримання прав адміністратора

1. Натисніть кнопку **⚙️ Налаштування**
2. Увімкніть **Режим адміністратора**
3. Введіть PIN-код: `1111`
